# FP_Group 6_Mahdi-Mohammadkhani_ITAI2372
Conceptual design plan for AI solution for NASA space mission
Track Chosen: Conceptual Design
Project Description:
This project is a conceptual design plan for an AI solution supporting a NASA space mission. It will identify a specific mission challenge, propose an AI-based solution, outline a testing plan, and describe steps for implementation.

How to Read This Documentation:
This README provides an overview of the project. It will guide you through the project goals, design approach, and testing plan.

This proposal includes a clear definition of the mission challenge, a step-by-step AI design plan, a comprehensive testing strategy, and expected outcomes. Next steps will involve research into AI tools, defining success metrics, and preparing a presentation.

Mission Challenge: This project focuses on an Earth observation NASA mission aimed at monitoring deforestation and urban expansion over time. The challenge is that rapid deforestation and unplanned urban growth are difficult to track consistently using traditional methods. By leveraging satellite imagery and AI, we aim to detect and quantify these changes automatically
